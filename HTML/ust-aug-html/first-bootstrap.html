Container Tags

  1. Header

    — defines a header for the document or a section that’s typically at the top of the page and meant to be introductory, which might contain heading elements, a logo, a search form, etc.

  2. Footer

    — defines a footer for the document or a section, which might contain author information, links to related pages, etc.

  3. Nav

    — defines a set of navigation links in the document. Commonly used for menus, tables of content, and indexes

  4. Main

    — defines the main content of the document. The Main element should not contain repeated content like navigation links, site logos, search forms, etc.

  5. Section

    — defines a section of the document that groups content with similar themes together. Webflow’s section element has a Section HTML5 tag by default.

  6. Article

    — defines a stand-alone composition on a page, like a forum post, newspaper article, or blog post

  7. Aside

    — defines content aside from the page content, like a sidebar or a call-out

  8. Address

    — defines the contact information for the author/owner of a document or an article‍

  9. Figure

    — defines self-contained content (e.g., illustrations, diagrams, photos, code blocks, etc.)
